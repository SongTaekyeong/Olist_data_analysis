# 🛒 Olist 이커머스 고객 이탈 분석
> 📝 본 프로젝트는 현재 통합·고도화 작업 중입니다. 
> Sensitivity analysis 검증, 통계 검정, 예측 모델링을 추가한 업데이트 버전을 준비 중입니다.


## 📌 프로젝트 개요
브라질 이커머스 플랫폼 Olist 데이터를 활용하여
고객 이탈 패턴을 분석하고 재구매 유도를 위한 타겟 고객을 도출하는 프로젝트입니다.

- **분석 환경:** Google BigQuery
- **데이터 출처:** [Kaggle - Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- **분석 기간:** 2016년 ~ 2018년

## 🎯 이탈 고객 정의
| 항목 | 내용 |
|------|------|
| 이탈(1) | 재구매 이력이 없는 고객 |
| 비이탈(0) | 재구매 이력이 있는 고객 |
| 근거 | 재구매 고객의 평균 재구매 간격이 87일로, 이 기간이 지나도 재구매가 없는 고객은 사실상 이탈로 판단 |
| 분석 대상 | delivered 상태의 주문 고객 |
---
초기 분석에서 이탈률이 97%로 매우 높게 나와 분석 정의의 타당성을 의심했습니다. 재구매 간격 분포를 분석해 p75(121일)와 p90(240일) 두 기준으로 sensitivity analysis를 수행한 결과, 표본을 46% 줄여도 이탈률은 96.03%로 유지되었습니다. 이를 통해 96%대 이탈률이 분석 정의의 artifact가 아닌, Olist 마켓플레이스의 구조적 특성임을 확인했습니다.


## 📊 분석 결과

### 1. 이탈 현황
| 구분 | 인원 | 비율 |
|------|------|------|
| 이탈(1) | 90,557명 | 약 97% |
| 비이탈(0) | 2,801명 | 약 3% |
| 전체 | 93,358명 | 100% |

### 2. 결제금액 분석
| 구분 | 평균 결제금액 |
|------|------|
| 이탈 고객 | $159 |
| 비이탈 고객 | $143 |

> 이탈 고객이 비이탈 고객보다 평균 결제금액이 높음
> 고가 상품 구매 후 재구매로 이어지지 않는 패턴 확인

### 3. 카테고리별 분석
| 순위 | 이탈 고객 | 비이탈 고객 |
|------|------|------|
| 1위 | cama_mesa_banho | cama_mesa_banho |
| 2위 | beleza_saude | moveis_decoracao |
| 3위 | esporte_lazer | esporte_lazer |

> 뷰티/건강(beleza_saude) 카테고리는 소모품 특성상
> 재구매 주기가 짧아야 하나 이탈률이 높음
> 경쟁사로의 이탈 가능성이 높은 카테고리

### 4. 지역별 분석
| 순위 | 지역 | 위치 |
|------|------|------|
| 1위 | CE (세아라) | 북동부 |
| 2위 | RN (히우그란지두노르치) | 북동부 |
| 3위 | MA (마라냥) | 북동부 |

> 이탈률 상위 지역이 모두 브라질 북동부에 집중
> 배송 인프라 부족으로 인한 만족도 저하 가능성

## 🎯 최종 인사이트 및 액션 플랜

### 핵심 발견
1. 전체 고객의 **97%가 재구매 없이 이탈**
2. 이탈 고객의 평균 결제금액($159)이 비이탈보다 높음
3. **뷰티/건강 카테고리**에서 이탈 집중
4. **북동부 지역(CE, RN, MA)** 이탈률 높음

### 액션 플랜
| 타겟 | 전략 |
|------|------|
| 고액 결제 이탈 고객 23,481명 | 우선 재구매 쿠폰 발행 |
| 뷰티/건강 카테고리 이탈 고객 | 카테고리 전용 할인 쿠폰 제공 |
| 북동부 지역 이탈 고객 | 배송 개선 + 쿠폰 병행 전략 |
